Chart Tests Plugin

Q 1: What is included in chart unit tests plugin?

Unit Tests

Unit tests collection will try to cover all complex logic of every class in chart related packages. It devotes to test the methods containing complicated selection structure and repetition structure.

Acceptance Tests

Acceptance tests will give the code quality control from function view. It checks whether the interface could work properly, also verifies whether the relevant function classes could work together to give expected outputs / results.

Smoke Tests

Smoke tests provide a minimal set of test cases which are required to make sure the chart work properly. Chart smoke tests ensure that the basic operations on chart could be processed.

Regression Tests

Regression tests include the selective tests collection of chart related functions, which ensure that no previously fixed functions will fail and newly added features have not created problems with previous version of the code. In this collection, all the test cases are selected from the resolved bugs in Tracker or Bugzilla. And the test suite will be expanded to catch up the expansion of bugzilla.
